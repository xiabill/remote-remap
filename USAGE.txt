Remote Remap 使用说明
======================

▌安装

1. 把 RemoteRemap.app 拖到「应用程序」文件夹
2. 启动一次（在「应用程序」里双击，或在 Launchpad 找到）
3. 第一次会被 macOS 拦下（"未受信任的开发者"），绕过：
   - 系统设置 → 隐私与安全性 → 拉到最下 → 点「仍要打开 RemoteRemap」
   - 或：访达 → 应用程序 → 右键 RemoteRemap → 打开 → 仍要打开

▌授权（重要）

启动后会依次弹两个权限对话框，两个都要给：

  1) "RemoteRemap 想要监听键盘事件"
     → 系统设置 → 隐私与安全性 → 输入监听 → 打开 RemoteRemap

  2) "RemoteRemap 想要控制您的电脑"
     → 系统设置 → 隐私与安全性 → 辅助功能 → 打开 RemoteRemap

两个权限都打开后，杀掉 RemoteRemap 重新启动一次（让权限生效）：

  在菜单栏图标上右键 → 退出
  然后再去应用程序里打开

▌使用

- 菜单栏会出现一个遥控器图标 📺
- 左键图标 → 配置面板
- 右键图标 → 快捷菜单（暂停 / 重启 / 退出等）

▌配置面板

每颗按键独占一行：

  [☑] 按键名  [目标键 ▼]  [+]

  - 勾选：启用该按键的映射
  - 目标键：选你想要触发的键
  - +：追加 chord 按键（变成多键组合，例如 ⌘ ⇧ V）

▌长按自动重复（可调 / 可关）

按住遥控器按键不放时，会先立刻触发一次映射，
等过启动延迟后开始按重复间隔自动 fire chord，松手就停。

在配置面板底部「长按自动重复」区里可以：
  - 总开关：彻底关掉，所有按键都只能"按一次触发一次"
  - 启动延迟：200–1500 ms，默认 500 ms
  - 重复间隔：30–500 ms，默认 100 ms（约 10 次/秒）

▌已知限制

🎤 语音键的系统听写无法完全拦截（macOS 内核私有路由），
   解决：系统设置 → 键盘 → 听写 → 把快捷键改成「关闭」

🎙️ 遥控器麦克风录音 1–2 分钟后会自动断流（遥控器固件省电
   策略，本 app 无法干预）。建议在 Typeless / Whisper 这类
   工具里把输入设备改为 Mac 自带麦克风，遥控器只做触发。

▌状态栏图标颜色

  绿点 = 运行中 + 已连接遥控器
  橙点 = 运行中但找不到遥控器（接收器没插？）
  灰   = 已暂停

▌支持的遥控器

只认 VID 0x1915 / PID 0x1025 的 2.4G USB 接收器
（XING WEI / 国产 Android TV 风格迷你遥控键盘常见方案）
